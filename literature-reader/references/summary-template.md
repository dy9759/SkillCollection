# 文献概要模板

生成文献概要时，严格按照以下结构输出 Markdown。根据论文实际内容填写，缺失部分标注"未提及"。

---

```markdown
# 文献概要：[论文完整标题]

## 基本信息

| 项目 | 内容 |
|------|------|
| **标题** | [中英文标题] |
| **作者** | [第一作者, 通讯作者(*), 等] |
| **机构** | [第一作者所在机构] |
| **发表** | [期刊/会议名称, 年份, 卷号/页码] |
| **DOI/链接** | [如有] |
| **关键词** | [论文关键词, 3-6个] |

## 一句话总结

> [用一句话概括论文的核心贡献，格式：本文提出/发现/验证了……，用于/解决了……问题]

## 研究背景与动机

- **领域背景**：[该研究所在领域的简要背景，2-3句]
- **现有问题**：[当前方法/认知存在的不足或空白]
- **研究动机**：[为什么要做这项研究，要解决什么问题]

## 核心贡献

1. [贡献1：简明扼要]
2. [贡献2]
3. [贡献3，如有]

## 方法与技术路线

### 整体思路

[用2-3句话描述方法论的核心思路]

### 关键步骤

1. **[步骤/模块名称]**：[描述]
2. **[步骤/模块名称]**：[描述]
3. **[步骤/模块名称]**：[描述]

### 关键公式/算法（如有）

[列出论文中最核心的1-2个公式或算法思想，用自然语言解释其含义]

## 实验与结果

### 实验设置

- **数据集**：[使用的数据集名称及规模]
- **基线方法**：[对比的方法]
- **评估指标**：[使用的指标]

### 主要结果

| 方法 | 指标1 | 指标2 | 指标3 |
|------|-------|-------|-------|
| Baseline A | - | - | - |
| Baseline B | - | - | - |
| **本文方法** | **-** | **-** | **-** |

[1-2句话总结结果亮点，标注出处如 "见 Table X"]

### 消融实验（如有）

[简述消融实验的关键发现]

## 局限性与未来工作

- **局限**：[作者明确提到的不足]
- **未来方向**：[作者提出的后续工作方向]

## 个人评价与思考

- **优点**：[从方法创新性、实验充分性、写作质量等维度评价]
- **不足**：[可改进之处]
- **启发**：[对自己研究/工作的启示，可以与哪些方向结合]
- **相关文献**：[论文中引用的值得进一步阅读的关键文献，列出2-3篇]

---

*概要生成日期：[YYYY-MM-DD]*
```

## 横向对比表模板（批量阅读时使用）

批量阅读多篇文献后，在所有概要之后输出以下对比表：

```markdown
# 文献横向对比

| 论文 | 年份 | 核心方法 | 数据集 | 关键指标 | 主要贡献 |
|------|------|----------|--------|----------|----------|
| [简称1] | 20XX | ... | ... | ... | ... |
| [简称2] | 20XX | ... | ... | ... | ... |

## 关系分析

- **互补关系**：[哪些论文从不同角度解决同一问题]
- **递进关系**：[哪些论文是前序工作的改进]
- **竞争关系**：[哪些论文在相同设置下直接对比]
```
