## 测试分类 2：发散搜索测试

### TC-005: 人物发散（销售场景）
**输入**: "帮我找一下张总的信息"
**预期发散维度**:
1. 直接搜索："张总"
2. 扩展搜索：
   - 别名扩展："张经理"、"张老板"
   - 关联维度：
     - 联系方式（名片、通讯录）
     - 近期沟通（会议记录、邮件）
     - 参与项目（ABC项目、合作案）
     - 相关团队（团队成员、合作伙伴）
     - 个人偏好（喜好、禁忌）

**预期搜索调用**:
```python
# 并行调用
search_notes(keyword="张总")
search_notes(keyword="张经理")
search_notes(keyword="张总", tags=["会议"])
search_notes(keyword="张总", tags=["项目"])
find_tags(keyword="张")
```
